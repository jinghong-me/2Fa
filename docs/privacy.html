隐私政策

最后更新时间：2026年4月24日

概述

身份验证助手（以下简称"我们"）非常重视用户的隐私保护。本隐私政策说明了我们如何收集、使用、存储和保护您的个人信息。使用我们的应用即表示您同意本政策中描述的做法。

核心原则

我们的承诺：

  • ✅ 不上云：所有数据都在您的手机本地处理，不会上传到我们的服务器
  • ✅ 不收集：不会收集您的个人信息、验证码密钥等敏感数据
  • ✅ 不追踪：不集成任何统计、分析或广告 SDK
  • ✅ 完全可控：所有权限和数据都由您自己掌控

信息收集与使用

验证码密钥

用途：仅在您的手机本地用于生成 TOTP 验证码

存储：安全保存在您手机的本地存储中，不会上传到任何服务器

重要：验证码密钥是您账号安全的关键，请妥善保管，不要泄露给他人！

账号信息

您添加的服务名称、账号名等信息保存在您手机的本地存储中，用于区分不同的验证账号，不会上传。

权限说明

  • 相机权限：用于扫描二维码添加验证账号（可选）
  • 网络权限：仅用于生成二维码时可能的临时文件操作，实际上不需要联网
  • 存储权限：用于保存二维码图片（可选）

所有权限都需要您主动授权，您可以随时在系统设置中撤销。

数据存储

本地存储

所有数据都存储在您手机的私有目录中，包括：

  • 验证码密钥和配置
  • 服务名称和账号信息
  • 应用设置

服务器存储

我们没有服务器存储您的数据！

应用是纯本地运行的工具，我们不收集、不存储、不上传任何用户数据。

您的权利

  • 查看数据：可以在应用内查看所有添加的验证账号
  • 删除数据：可以在应用内删除单个账号，或卸载应用删除所有数据
  • 导出备份：可以导出验证码二维码进行备份
  • 控制权限：可以在系统设置中随时授予或撤销权限

第三方服务

第三方 SDK

当前版本未集成任何第三方 SDK（包括统计、广告、崩溃分析等）。

安全建议

为了您的账号安全，建议您：

  • 不要将验证码截图发给他人
  • 定期备份验证码密钥
  • 不要在公共设备上使用此应用
  • 给手机设置锁屏密码

政策更新

我们可能会不时更新本隐私政策。重大变更时，我们会通过应用内通知或其他方式告知您。建议您定期查看本政策以了解最新信息。
